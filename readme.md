## Sobre este espacio

Via-Integra reúne textos de reflexión, práctica y pensamiento  
nacidos desde una trayectoria que integra el Kung Fu Shaolin,  
el Budismo Vajrayana, la Medicina Tradicional China,  
la teología racional y cincuenta años de trabajo con animales.

No es un compendio académico. Es un camino documentado.
